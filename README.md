# Cardatabase
Car API based on application developed in Juha Hinkula's book, [Hands-On Full Stack Development with Spring Booot 2.0 and React](https://www.amazon.com/Hands-Stack-Development-Spring-React/dp/1789138086/)

## Stack
- Java 8
- Maven
- Spring Boot
- Spring Data REST
- Spring Security
- Liquibase
- Travis CI
